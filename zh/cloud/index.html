    Nimbus Cloud

    托管的 Nimbus —— 我们负责运维,你专注于交付。

    你能获得什么

    • 几天内上线,而不是几个月。 我们在 24 小时内为你配置好 Nimbus 实例,让你的团队专注于构建工作流,而不是处理基础架构。
    • 每周省下 10 小时。 不再需要打补丁、升级 Docker、为这套栈轮流值班 —— 你的团队专注于只有他们能完成的工作。
    • 事故期间也能安心睡眠。 我们提供 24/7 监控,只有当需要人工决策时才会通知你。
    • 升级不会中断业务。 新版 Nimbus 零停机部署,每次变更都由你审批。

    工作方式

    1. 注册。 一次 15 分钟的通话,以确认适配性和范围。
    2. 我们负责配置。 在你的区域部署一个专属的 Nimbus 实例,配置好你的 MCP 服务器和 OAuth 连接。
    3. 你接入自己的工具。 同一个 Nimbus,同一些 MCP 服务器 —— 你这边无需学习任何新东西。

    定价

    按工作负载计费,随使用量伸缩。由于每个团队的使用情况不同,仅提供报价。

    联系我们获取报价 →

    安全与 SLA

    • 传输中与静态数据均加密
    • SOC 2 —— 正在推进(目标 2026 年 Q4)
    • 99.9% 可用性 SLA,如未达成则提供抵扣
    • 美国和欧盟区域,你的数据留在你指定的位置

    FAQ

    之后可以从 self-host 迁移到 cloud 吗? 可以。一键导入你的 mcp.json、.env 和 OAuth 令牌。我们会在一次通话中与你一起完成切换。

    可以从 cloud 迁回 self-host 吗? 可以。同一套一键导出功能可用于任何 Nimbus 安装。

    我的数据存储在哪里? 美国或欧盟,注册时由你选择。未经告知,我们不会迁移数据。

    你们能访问哪些内容? 你的 Nimbus 实例的运行日志,以及你的工作流生成的 MCP 服务器输出。我们不会读取你的源代码、CRM 或邮件内容。

    可以取消吗? 随时可以。你会保留数据导出。

    提供哪些区域? US-East、US-West、EU-Frankfurt。其他区域可按需提供。

    想要获得指导性演示?预约一次 30 分钟的通话,我们将一起梳理你的实例范围。
