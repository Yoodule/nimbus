## FAQ

**之后可以从 self-host 迁移到 cloud 吗?**
可以。一键导入你的 `mcp.json`、`.env` 和 OAuth 令牌。我们会在一次通话中与你一起完成切换。

**可以从 cloud 迁回 self-host 吗?**
可以。同一套一键导出功能可用于任何 Nimbus 安装。

**我的数据存储在哪里?**
美国或欧盟,注册时由你选择。未经告知,我们不会迁移数据。

**你们能访问哪些内容?**
你的 Nimbus 实例的运行日志,以及你的工作流生成的 MCP 服务器输出。我们不会读取你的源代码、CRM 或邮件内容。

**可以取消吗?**
随时可以。你会保留数据导出。

**提供哪些区域?**
US-East、US-West、EU-Frankfurt。其他区域可按需提供。
